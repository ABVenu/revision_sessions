41.

```js

  return;
  a + b;

```

> Output: `undefined` (due to automatic semicolon insertion)

---

42.

```js
let x = function fact(n) {
  if (n === 0) return 1;
  return n * fact(n - 1);
};
console.log(x(3));
```

> Output: `6` (named function expression, can recurse using `fact`)

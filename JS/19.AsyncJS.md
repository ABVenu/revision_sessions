### âœ… **Coding Snippet / Implementation Practice**

31. **Write a callback-based function that adds two numbers after 1 sec.**

```js
function add(a, b, cb) {
  setTimeout(() => {
    cb(a + b);
  }, 1000);
}
```

---

32. **Convert the above to a Promise-based function**

```js
function add(a, b) {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve(a + b);
    }, 1000);
  });
}
```

---

33. **Write async/await version of the above**

```js
async function run() {
  const result = await add(2, 3);
  console.log(result);
}
run();
```

---

34. **Demonstrate nested callbacks (callback hell)**

```js
setTimeout(() => {
  console.log("Step 1");
  setTimeout(() => {
    console.log("Step 2");
    setTimeout(() => {
      console.log("Step 3");
    }, 1000);
  }, 1000);
}, 1000);
```

37. **Async/await with try...catch**

```js
async function fetchData() {
  try {
    let res = await fetch("https://jsonplaceholder.typicode.com/posts");
    let data = await res.json();
    console.log(data.length);
  } catch (e) {
    console.log("Error:", e.message);
  }
}
```

38. **Race condition with `Promise.race()`**

```js
Promise.race([
  new Promise((res) => setTimeout(() => res("First"), 500)),
  new Promise((res) => setTimeout(() => res("Second"), 1000)),
]).then(console.log);
```

> First

39. **Delay utility using Promise**

```js
function delay(ms) {
  return new Promise((res) => setTimeout(res, ms));
}

async function main() {
  console.log("Waiting...");
  await delay(1000);
  console.log("Done!");
}
```
